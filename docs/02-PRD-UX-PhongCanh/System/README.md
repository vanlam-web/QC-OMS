## 1. Phạm vi

System trong QC-OMS chỉ làm phần cần để vận hành:

- tài khoản nhân viên
- phân quyền theo permission
- trạng thái active/inactive
- máy trạm/quầy POS

Không làm các module nhân sự KiotViet:

- chấm công
- lịch làm việc
- bảng lương
- hoa hồng
- KPI nhân viên
